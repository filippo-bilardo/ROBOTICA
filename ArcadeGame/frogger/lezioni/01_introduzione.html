Introduzione a JavaScript

Cos'è JavaScript?

JavaScript è un linguaggio di programmazione interpretato, orientato agli oggetti e di alto livello. È uno dei tre pilastri del web moderno, insieme a HTML (struttura) e CSS (stile). Mentre HTML e CSS sono linguaggi di markup che definiscono la struttura e l'aspetto di una pagina web, JavaScript è un vero e proprio linguaggio di programmazione che aggiunge interattività e comportamento dinamico alle pagine web.

Nonostante il nome, JavaScript non ha una relazione diretta con il linguaggio Java. Il nome fu scelto principalmente per ragioni di marketing, dato che Java era molto popolare quando JavaScript fu introdotto.

Nota: JavaScript è spesso abbreviato come "JS" e i file JavaScript hanno tipicamente l'estensione .js.

Storia di JavaScript

La storia di JavaScript è affascinante e riflette l'evoluzione del web stesso. Ecco i momenti chiave nella storia di questo linguaggio:

1995

Brendan Eich crea JavaScript (inizialmente chiamato "Mocha", poi "LiveScript") in soli 10 giorni mentre lavorava per Netscape. Il linguaggio viene rilasciato con Netscape Navigator 2.0.

1996

JavaScript viene rinominato da "LiveScript" a "JavaScript" come mossa di marketing per sfruttare la popolarità di Java. Nello stesso anno, Microsoft rilascia JScript, una versione compatibile di JavaScript per Internet Explorer.

1997

JavaScript viene standardizzato da ECMA International come ECMAScript (ECMA-262). Questa standardizzazione è fondamentale per garantire la compatibilità tra diversi browser.

2005

Jesse James Garrett conia il termine "AJAX" (Asynchronous JavaScript and XML), una tecnica che permette di aggiornare parti di una pagina web senza ricaricarla completamente. Questo segna l'inizio dell'era delle applicazioni web moderne.

2009

ECMAScript 5 (ES5) viene rilasciato, introducendo importanti miglioramenti come i metodi di array, il supporto JSON nativo e la modalità strict.

2015

ECMAScript 2015 (ES6) viene rilasciato, rappresentando il più grande aggiornamento del linguaggio fino ad oggi. Introduce classi, moduli, arrow functions, promesse e molto altro.

2016-Oggi

JavaScript continua a evolversi con rilasci annuali di nuove specifiche ECMAScript. Il linguaggio è ora utilizzato non solo per il web, ma anche per lo sviluppo di applicazioni desktop, mobile e server (Node.js).

Caratteristiche Principali di JavaScript

  • Linguaggio interpretato: JavaScript viene eseguito direttamente dal browser senza necessità di compilazione.
  • Tipizzazione dinamica: Le variabili non hanno un tipo fisso e possono cambiare tipo durante l'esecuzione.
  • Orientato agli oggetti: Supporta la programmazione orientata agli oggetti con prototipi anziché classi (anche se ES6 ha introdotto la sintassi delle classi).
  • Funzioni di prima classe: Le funzioni sono trattate come qualsiasi altra variabile e possono essere passate come argomenti, restituite da altre funzioni, ecc.
  • Event-driven: JavaScript è progettato per rispondere a eventi come clic del mouse, pressioni di tasti, caricamento di pagine, ecc.
  • Single-threaded con modello asincrono: JavaScript esegue un'operazione alla volta, ma può gestire operazioni asincrone tramite callback, promesse e async/await.

Perché JavaScript è Importante?

JavaScript è diventato uno dei linguaggi di programmazione più importanti e diffusi al mondo per diversi motivi:

  • Ubiquità: È supportato da tutti i browser web moderni senza necessità di plugin.
  • Versatilità: Può essere utilizzato per sviluppare applicazioni web, mobile, desktop e server.
  • Ecosistema ricco: Dispone di un vasto ecosistema di librerie, framework e strumenti (React, Angular, Vue, Node.js, ecc.).
  • Comunità attiva: Ha una delle comunità di sviluppatori più grandi e attive.
  • Evoluzione continua: Il linguaggio continua a evolversi e migliorare con nuove funzionalità.

JavaScript e lo Sviluppo di Giochi

JavaScript, combinato con HTML5 Canvas, è diventato una piattaforma potente per lo sviluppo di giochi web. Ecco alcuni vantaggi dell'utilizzo di JavaScript per lo sviluppo di giochi:

  • Accessibilità: I giochi JavaScript possono essere giocati su qualsiasi dispositivo con un browser, senza installazioni.
  • API Canvas: Fornisce un'API potente per disegnare grafica 2D (e con WebGL anche 3D).
  • Prestazioni: I moderni motori JavaScript sono ottimizzati per offrire prestazioni elevate.
  • Facilità di distribuzione: I giochi possono essere facilmente condivisi e distribuiti online.

Il Nostro Progetto: Frogger

In questo corso, utilizzeremo JavaScript per ricreare il classico gioco arcade Frogger. Questo progetto ci permetterà di esplorare molti concetti fondamentali di JavaScript in un contesto pratico e divertente.

Frogger è un gioco arcade del 1981 in cui il giocatore guida una rana attraverso una strada trafficata e un fiume pieno di pericoli per raggiungere la sicurezza. È un gioco perfetto per imparare JavaScript perché:

  • Ha una meccanica di gioco semplice ma coinvolgente
  • Richiede l'implementazione di concetti come il movimento, le collisioni e il punteggio
  • Può essere realizzato con grafica semplice utilizzando Canvas
  • Può essere esteso con funzionalità aggiuntive man mano che si acquisiscono nuove competenze

Il tuo primo codice JavaScript

Ecco un semplice esempio di codice JavaScript che mostra un messaggio di benvenuto:

// Questo è un commento in JavaScript

// Definizione di una variabile
let nomeGioco = "Frogger";

// Funzione che mostra un messaggio di benvenuto
function mostraBenvenuto() {
    console.log("Benvenuto al corso di " + nomeGioco + "!");
    alert("Benvenuto al corso di " + nomeGioco + "!");
}

// Chiamata della funzione
mostraBenvenuto();

Questo codice definisce una variabile, crea una funzione e la chiama per mostrare un messaggio di benvenuto sia nella console che in un alert.
