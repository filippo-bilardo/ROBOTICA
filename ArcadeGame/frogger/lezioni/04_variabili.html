Variabili e Tipi di Dati

Introduzione

Le variabili sono uno dei concetti fondamentali in qualsiasi linguaggio di programmazione, incluso JavaScript. Esse ci permettono di memorizzare e manipolare dati all'interno del nostro programma. In questa lezione, esploreremo come dichiarare e utilizzare variabili in JavaScript, nonché i diversi tipi di dati che possono essere memorizzati in esse.

Cos'è una Variabile?

Una variabile in JavaScript è un contenitore per memorizzare valori di dati. Puoi immaginare una variabile come una scatola etichettata: l'etichetta è il nome della variabile, e ciò che è contenuto nella scatola è il valore della variabile.

In JavaScript, le variabili sono dinamicamente tipizzate, il che significa che una variabile può contenere diversi tipi di dati durante l'esecuzione del programma e non è necessario dichiarare esplicitamente il tipo di dati che conterrà.

Dichiarazione di Variabili

In JavaScript moderno (ES6+), ci sono tre modi principali per dichiarare variabili:

1. Usando let

let è il modo moderno per dichiarare variabili che possono cambiare valore nel tempo. 

let punteggio = 0;
punteggio = 10; // Il valore può essere cambiato

let nomeGiocatore = "Giocatore1";
let inGioco = true;

2. Usando const

const è usato per dichiarare variabili il cui valore non cambierà (costanti). 

const VITE_INIZIALI = 3;
// VITE_INIZIALI = 4; // Questo causerebbe un errore

const LARGHEZZA_CANVAS = 600;
const ALTEZZA_CANVAS = 600;

3. Usando var (modo tradizionale)

var è il modo tradizionale per dichiarare variabili in JavaScript, ma ha alcune peculiarità riguardo allo scope che possono causare confusione. Nei progetti moderni, è generalmente consigliato utilizzare let e const invece di var. 

var vecchioPunteggio = 0;
vecchioPunteggio = 20; // Il valore può essere cambiato
Nota: La principale differenza tra let e var riguarda lo scope. Le variabili dichiarate con var hanno scope di funzione, mentre quelle dichiarate con let hanno scope di blocco. Questo significa che una variabile let dichiarata all'interno di un blocco (come un ciclo for o un'istruzione if) è accessibile solo all'interno di quel blocco.

Tipi di Dati in JavaScript

JavaScript ha diversi tipi di dati che possono essere memorizzati in variabili. I tipi di dati principali sono:

Tipo Descrizione Esempio
Number Rappresenta numeri interi e decimali let velocita = 5;
let posizione = 10.5;
String Rappresenta testo let messaggio = "Game Over";
let nome = 'Frogger';
Boolean Rappresenta valori vero/falso let inGioco = true;
let collisione = false;
Undefined Rappresenta una variabile dichiarata ma non inizializzata let prossimoPunteggio;
Null Rappresenta l'assenza intenzionale di un valore let giocatore = null;
Object Rappresenta una collezione di dati correlati let rana = {x: 300, y: 550, larghezza: 30, altezza: 30};
Array Rappresenta una lista ordinata di valori let ostacoli = [auto1, auto2, tronco1, tronco2];

Utilizzo delle Variabili nel Gioco Frogger

Nel nostro gioco Frogger, utilizzeremo variabili per memorizzare diversi tipi di informazioni, come:

  • La posizione della rana (giocatore)
  • Le posizioni degli ostacoli (auto, tronchi, ecc.)
  • Il punteggio del giocatore
  • Il numero di vite rimanenti
  • Lo stato del gioco (in corso, in pausa, game over)

Esempio: Variabili per il Giocatore

// Posizione iniziale della rana
let ranaX = 300; // Centro orizzontale del canvas
let ranaY = 550; // Parte inferiore del canvas

// Dimensioni della rana
const RANA_LARGHEZZA = 30;
const RANA_ALTEZZA = 30;

// Stato del giocatore
let vite = 3;
let punteggio = 0;
let inGioco = false;

Esempio: Variabili per gli Ostacoli

// Array di ostacoli (auto)
let auto = [
    {x: 0, y: 450, larghezza: 50, altezza: 30, velocita: 2},
    {x: 150, y: 450, larghezza: 50, altezza: 30, velocita: 2},
    {x: 300, y: 400, larghezza: 50, altezza: 30, velocita: -3},
    {x: 450, y: 400, larghezza: 50, altezza: 30, velocita: -3},
    {x: 100, y: 350, larghezza: 50, altezza: 30, velocita: 1.5}
];

// Array di piattaforme (tronchi)
let tronchi = [
    {x: 0, y: 250, larghezza: 100, altezza: 30, velocita: 1},

    {x: 100, y: 150, larghezza: 100, altezza: 30, velocita: 2}
];

Scope delle Variabili

Lo scope di una variabile determina dove quella variabile può essere accessibile nel codice. In JavaScript, ci sono principalmente due tipi di scope:

  • Scope Globale: Le variabili dichiarate fuori da qualsiasi funzione o blocco sono accessibili ovunque nel codice.
  • Scope Locale: Le variabili dichiarate all'interno di una funzione o un blocco sono accessibili solo all'interno di quella funzione o blocco.

Esempio: Scope delle Variabili

// Variabile globale
let punteggio = 0;

function aggiornaPunteggio() {
    // Variabile locale
    let puntiGuadagnati = 10;
    
    // Possiamo accedere alla variabile globale
    punteggio += puntiGuadagnati;
    
    if (punteggio > 100) {
        // Variabile con scope di blocco
        let messaggio = "Nuovo record!";
        console.log(messaggio);
    }
    
    // Questo causerebbe un errore perché 'messaggio' è fuori scope
    // console.log(messaggio);
}

// Possiamo accedere alla variabile globale
console.log(punteggio);

// Questo causerebbe un errore perché 'puntiGuadagnati' è fuori scope
// console.log(puntiGuadagnati);

Best Practices per l'Uso delle Variabili

  • Usa nomi descrittivi: I nomi delle variabili dovrebbero descrivere chiaramente ciò che contengono.
  • Usa camelCase: In JavaScript, la convenzione è di usare camelCase per i nomi delle variabili (es. nomeGiocatore, punteggioTotale).
  • Usa UPPERCASE per le costanti: Le costanti sono spesso scritte in maiuscolo con underscore (es. MAX_VITE, VELOCITA_GIOCO).
  • Preferisci const quando possibile: Se una variabile non deve cambiare valore, usa const per renderlo esplicito.
  • Evita variabili globali: Limita l'uso di variabili globali per evitare conflitti e rendere il codice più modulare.
