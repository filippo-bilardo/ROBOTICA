Struttura Base del Progetto

Introduzione

Una buona organizzazione del progetto è fondamentale per lo sviluppo di qualsiasi applicazione, incluso il nostro gioco Frogger. In questa lezione, esploreremo in dettaglio la struttura del progetto che abbiamo introdotto nella lezione precedente e creeremo i file di base necessari per iniziare a sviluppare il gioco.

Panoramica della Struttura del Progetto

Ecco la struttura completa del nostro progetto Frogger:

frogger/ ├── index.html # Pagina principale del gioco ├── css/ │ └── style.css # Fogli di stile ├── js/ │ ├── game.js # Logica principale del gioco │ ├── player.js # Classe per il giocatore (rana) │ ├── obstacle.js # Classe per gli ostacoli │ └── utils.js # Funzioni di utilità └── assets/ ├── images/ # Immagini del gioco └── sounds/ # Effetti sonori

Spiegazione dei File e delle Cartelle

1. File HTML (index.html)

Il file index.html è il punto di ingresso del nostro gioco. Contiene la struttura HTML di base, i collegamenti ai file CSS e l'inclusione dei file JavaScript. 

<!DOCTYPE html>
<html lang="it">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Frogger - Gioco JavaScript</title>
    <link rel="stylesheet" href="css/style.css">
</head>
<body>
    <h1>Frogger</h1>
    <div id="game-container">
        <canvas id="game-canvas" width="600" height="600"></canvas>
    </div>
    <div id="game-info">
        <p>Punteggio: <span id="score">0</span></p>
        <p>Vite: <span id="lives">3</span></p>
        <button id="start-button">Inizia Gioco</button>
    </div>
    
    <script src="js/utils.js"></script>
    <script src="js/player.js"></script>
    <script src="js/obstacle.js"></script>
    <script src="js/game.js"></script>
</body>
</html>

Nota che abbiamo aggiunto un elemento <canvas> all'interno del contenitore del gioco. Canvas è un elemento HTML5 che ci permette di disegnare grafica e animazioni utilizzando JavaScript. Sarà la base del nostro gioco Frogger.
